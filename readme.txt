Hide any block with conditions for the Gutenberg editor.

== Description ==

A simple plugin to help you hide any block on a page with conditions.

Features:

* Hide blocks based on device screen size.
* Hide blocks when positioned as the first or last child element.
* Hide blocks when the query block has no result

== FAQ ==

= Will it slow down my site? =

No, This plugin is optimized for lightweight and low resource consumption.
